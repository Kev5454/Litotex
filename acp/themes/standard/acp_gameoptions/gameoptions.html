Hier gibt es die Möglichkeit neue Statische Variablen für das Spiel zu konfigurieren.
Diese stehen im gesammten Spiel zur Verfügung.

Konfigurieren könnt ihr diese im Menüpunkt eurer Wahl.
Diese Funktion sollte mir Bedacht angewendet werden !

Beispiel (an Hand einer bereits existierenden Variable) :
Variablenname : op_set_offline
Variablentitel : Wartungsmodus
Type: Ja/ Nein
Anzeige im Menüpunkt: Wartungsmodus

Durch die Abfrage im Code der Variablen $op_set_offline kannst du bestimmen ob das Spiel im Offlinemodus ist.
Die Einstellungen für Online/ Offlinemodus können im Menüpunkt 'Wartungsmodus' eingestellt werden.

WICHTIG !!!: Zur besseren Übersicht müssen alle variablen mit op_ beginnen !
